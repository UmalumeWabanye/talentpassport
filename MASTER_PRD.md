# Talent Passport - Master PRD

## Purpose
This document defines the long-term product vision, core mission, and non-changing product principles for Talent Passport.

It is intentionally stable and high-level.
Detailed implementation scope must live in phase-specific PRDs under `phases/`.

## Product Vision
Talent Passport is a multi-tenant employability platform that helps youth showcase verified capability and helps organizations, employers, and stakeholders make better talent decisions.

## Product Outcomes
- Improve youth employability readiness through structured digital identity and evidence.
- Enable organizations and employers to evaluate candidates more effectively.
- Provide trusted, scalable infrastructure for long-term ecosystem growth.

## Scope Governance
- Delivery is phase-locked.
- Each phase is implemented only from its corresponding PRD.
- Future-phase capabilities must not be implemented early.

## Document Ownership
- This document changes infrequently.
- Use `ROADMAP.md` for phased planning.
- Use `ARCHITECTURE.md` for technical structure.
- Use `DEVELOPMENT_RULES.md` for permanent execution rules.
