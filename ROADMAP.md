# Talent Passport - Roadmap

## Planning Model
The roadmap is intentionally high-level. Execution details belong to phase PRDs in `phases/`.

## Phases
1. Phase 01 - Foundation
2. Phase 02 - Youth Portal
3. Phase 03 - Organisation
4. Phase 04 - AI
5. Additional phases to be defined

## Delivery Rules
- Only one phase is active at a time.
- No implementation from future phases during an active phase.
- Phase transition requires completion against the phase Definition of Done.
